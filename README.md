# 📊 Google Play Store Data Analysis

## 📌 Objective
Analyze Google Play Store data to identify factors influencing app success (ratings, installs, categories).

## 🧠 Key Questions
- Which categories have highest installs?
- Do paid apps perform better than free apps?
- What factors affect app ratings?

## 📊 Key Insights
- Free apps dominate installs
- Game & Tools categories have highest engagement
- Smaller app size correlates with higher installs

## 🛠️ Tech Stack
- Python (Pandas, NumPy)
- Matplotlib / Seaborn
- Jupyter Notebook

## 📂 Dataset
Kaggle Google Play Store dataset

## 🚀 Conclusion
This project demonstrates data cleaning, EDA, and insight extraction from real-world datasets.
